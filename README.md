# Customer Churn Analytics Dashboard
# Project Overview

This project analyzes customer churn behavior in a telecom company using Excel and Power BI.

The objective is to identify the major reasons behind customer churn and provide business recommendations to improve customer retention.

# Tools Used
Excel
Power BI
Pivot Tables
DAX
Dataset Information
Total Customers: 7043
Total Churn Customers: 1869
Churn Rate: 26.54%

# Key Insights
1. Contract-wise Churn
Month-to-month contracts contribute to 88.55% of total churn.
Customers with long-term contracts are less likely to churn.
2. Internet Service Analysis
Fiber optic customers account for 69.4% of churned customers.
DSL customers have significantly lower churn.
3. Monthly Charges Analysis
Churned customers pay higher average monthly charges.
Average Monthly Charges:
Churned Customers     : 74.44

Non-Churn Customers   : 61.27
Business Recommendations
Offer discounts for Month-to-month customers.
Introduce loyalty programs.
Improve Fiber optic customer satisfaction.
Encourage long-term contracts.
